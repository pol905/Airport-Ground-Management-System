-----------------Installation------------------

1.install python 3.7.x

2.install pip -> pytthon install pip

3. Open Command Prompt and change directory into root folder of the project

4. Run the below command in the terminal

cmd>pip install -r requirements.txt

5. Change Directory to the airport folder 

-----------------------------------------------------------------------------------------------------------

Running the app-

1. Run "python manage.py runserver" (without quotes) to start the server

2. Copy the local host IP address and visit it using a browser.

-----------------------------------------------------------------------------------------------------------

To read the code with proper formats use sublime text to open the entire project folder(airport)
